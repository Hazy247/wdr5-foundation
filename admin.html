How saving works

This site is hosted on GitHub Pages, so there is no built-in username/password login. The private GitHub update key is what gives permission to publish.

  1. Enter your GitHub repository and private update key below.
  2. Select Connect and load from GitHub.
  3. Edit homepage numbers or update posts.
  4. Select Save draft on the section you changed.
  5. Select Publish live changes to GitHub. That is the step that updates the public website.
Important: visitors can open this page if they know the address, but they cannot publish changes unless they have your private GitHub update key. Do not share that key publicly.
